# Teach Loop course instructions

This repository is a local-first tutoring workspace. The installed Teach Loop
skill is the source of truth for teaching behavior.

Before changing or creating a lesson:

1. Read the workspace state files at the repository root.
2. Read recent relevant lesson returns and learning records.
3. Re-open the sources supporting the target lesson.
4. Prefer components exported from `@/lesson-kit`.
5. Use stable, descriptive component IDs.
6. Keep factual claims cited inline with `<Cite source="..." />`.
7. Run validation, tests, and a production build.

Do not convert the learning map into a fixed syllabus. Do not infer mastery from
coverage or correctness after heavy assistance. Do not fabricate citations.
